# Habit Tracker

A Django-based web application for tracking daily and weekly habits, with social features to follow other users.

---

## Features

- [x] User Authentication with JWT - Secure user registration and login using JSON Web Tokens.
- [x] Habit Management - Create, view, update, and delete habits.
- [x] Habit Logging - Log daily completions for habits.
- [x] Streak Calculation - Automatically calculates consecutive day streaks for habits.
- [x] Tagging - Assign multiple tags to habits for better organization.
- [x] Social Features - Follow other users and view their public habits.
- [ ] Real-time chatting with other users you follow

# Details
---
- Uses the Django framework for python (backend).
- Frontend is rendered using Django Templates with vanilla CSS.
